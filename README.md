# Website Project

## Getting Started

### Prerequisites
- Node.js (recommend version 18 or later)
- npm

### Installation
```
npm install
```

### Development Commands
```
# Build the site
npx vitepress build docs

# Preview the built site locally
npx vitepress preview docs
```

### Project Structure
This project uses VitePress for documentation and website building.

### Documentation
For more information about VitePress configuration and features, refer to the official documentation:
- [VitePress Reference](https://vitepress.dev/reference)
