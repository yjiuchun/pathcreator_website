# VitePress 网站添加内容指南

## 📁 项目结构说明

```
web/pathcreator_website/
├── docs/                    # 📝 所有网站内容都在这里
│   ├── index.md            # 首页
│   ├── *.md                # 其他页面（Markdown文件）
│   ├── public/             # 🖼️ 图片等静态资源
│   │   ├── *.png          # 图片文件
│   │   ├── *.jpg
│   │   ├── *.svg
│   │   └── ...
│   └── .vitepress/        # 配置文件（一般不需要修改）
│       ├── config/        # 导航和侧边栏配置
│       └── theme/         # 主题样式
└── package.json
```

---

## 📝 如何添加文字内容

### 方法1：修改现有页面

直接编辑 `docs/` 目录下的 `.md` 文件：

- **首页**：编辑 `docs/index.md`
- **其他页面**：编辑对应的 `.md` 文件

### 方法2：创建新页面

1. **在 `docs/` 目录下创建新的 `.md` 文件**

   例如：创建 `docs/关于我们.md` 或 `docs/about.md`

2. **编写内容（使用 Markdown 语法）**

   ```markdown
   # 关于我们
   
   这是我们的公司介绍...
   
   ## 公司历史
   
   公司成立于...
   
   ## 联系方式
   
   - 电话：xxx
   - 邮箱：xxx
   ```

3. **在配置文件中添加导航链接**（可选）

   编辑 `docs/.vitepress/config/zh.ts`，在 `nav()` 函数中添加：

   ```typescript
   {
     text: '关于我们',
     link: '/关于我们'
   }
   ```

## 🖼️ 如何添加图片

### 方法1：使用 public 目录（推荐）

1. **将图片放到 `docs/public/` 目录**

   ```
   docs/public/
   ├── logo.png
   ├── company-photo.jpg
   └── products/
       └── robot-arm.png
   ```

2. **在 Markdown 文件中引用图片**

   ```markdown
   <!-- 引用 public 目录下的图片 -->
   ![公司Logo](/logo.png)
   
   <!-- 带说明文字 -->
   ![产品图片](/products/robot-arm.png)
   
   <!-- HTML 方式（可以设置大小） -->
   <img src="/logo.png" alt="公司Logo" width="200" />
   ```

   **注意**：路径以 `/` 开头，直接指向 `public/` 目录

### 方法2：使用相对路径（适合页面专用图片）

1. **在页面同目录下创建图片文件夹**

   ```
   docs/
   ├── 产品介绍.md
   └── products/
       └── robot-arm.png
   ```

2. **使用相对路径引用**

   ```markdown
   ![产品图片](./products/robot-arm.png)
   ```

## 📋 完整示例

### 示例1：创建"产品介绍"页面

1. **创建文件** `docs/产品介绍.md`

   ```markdown
   # 产品介绍
   
   ## 3D视觉系统
   
   ![3D视觉产品](/products/3d-vision.jpg)
   
   我们的3D视觉系统具有以下特点：
   - 高精度识别
   - 实时处理
   - 广泛应用
   
   ## 机械臂
   
   ![机械臂产品](/products/robot-arm.jpg)
   
   我们的机械臂产品...
   ```

2. **添加图片到 `docs/public/products/`**

   ```
   docs/public/products/
   ├── 3d-vision.jpg
   └── robot-arm.jpg
   ```

3. **在导航中添加链接**（编辑 `docs/.vitepress/config/zh.ts`）

   ```typescript
   function nav(): DefaultTheme.NavItem[] {
     return [
       {
         text: '产品介绍',
         link: '/产品介绍'
       },
       // ... 其他导航项
     ]
   }
   ```

### 示例2：修改首页内容

编辑 `docs/index.md`：

```markdown
---
layout: home

hero:
  name: "上海创路者机器人有限公司"
  text: "机器人外包之王"
  tagline: 3D视觉 机械臂 机器人底盘
  image:
    src: /company-logo.png    # 添加Logo图片
    alt: 公司Logo
  actions:
    - theme: brand
      text: 产品介绍
      link: /产品介绍
    - theme: alt
      text: 联系我们
```

## 📌 重要提示

1. **图片格式**：推荐使用 `.png`、`.jpg`、`.svg` 格式
2. **图片大小**：建议压缩图片，避免文件过大影响加载速度
3. **文件命名**：使用英文或拼音，避免中文文件名（可能导致问题）
4. **路径规则**：
   - `public/` 目录下的文件：使用 `/文件名` 
   - 相对路径：使用 `./文件夹/文件名`
5. **保存后**：修改文件后记得保存，VitePress 会自动检测变化
