## 参与开源

我们欢迎社区成员参与我们的开源项目：

1. **Star 项目**：如果项目对你有帮助，请给我们一个 Star
2. **提交 Issue**：发现问题或有功能建议，欢迎提交 Issue
3. **贡献代码**：提交 Pull Request，一起完善项目
4. **分享使用经验**：在 Discussions 中分享你的使用案例

---

## 开源协议

我们的开源项目遵循以下协议：

- **MIT License**：大多数项目使用 MIT 协议
- **Apache 2.0**：部分企业级项目使用 Apache 2.0 协议

请在使用前仔细阅读项目许可证。
