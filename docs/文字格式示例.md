# 文字格式控制指南

在 VitePress 的 features 或其他 Markdown 内容中，可以使用 HTML 标签来控制文字格式。

## 基本方法：使用 `<p>` 标签和内联样式

```html
<p style="font-size: 16px; color: #333; text-align: center;">
  你的文字内容
</p>
```

## 常用样式属性

### 字体大小 (font-size)
```html
<p style="font-size: 12px;">小号文字</p>
<p style="font-size: 16px;">正常文字</p>
<p style="font-size: 20px;">大号文字</p>
<p style="font-size: 24px;">超大号文字</p>
```

### 颜色 (color)
```html
<p style="color: #333;">深灰色</p>
<p style="color: #666;">中灰色</p>
<p style="color: #ff0000;">红色</p>
<p style="color: #0066cc;">蓝色</p>
<p style="color: #00aa00;">绿色</p>
```

### 字体粗细 (font-weight)
```html
<p style="font-weight: normal;">正常</p>
<p style="font-weight: bold;">粗体</p>
<p style="font-weight: 300;">细体</p>
<p style="font-weight: 600;">半粗体</p>
```

### 文字对齐 (text-align)
```html
<p style="text-align: left;">左对齐</p>
<p style="text-align: center;">居中</p>
<p style="text-align: right;">右对齐</p>
```

### 行高 (line-height)
```html
<p style="line-height: 1.5;">正常行高</p>
<p style="line-height: 2;">宽松行高</p>
<p style="line-height: 1.2;">紧凑行高</p>
```

### 字体样式 (font-style)
```html
<p style="font-style: normal;">正常</p>
<p style="font-style: italic;">斜体</p>
```

### 文字装饰 (text-decoration)
```html
<p style="text-decoration: underline;">下划线</p>
<p style="text-decoration: line-through;">删除线</p>
<p style="text-decoration: none;">无装饰</p>
```

## 组合使用示例

### 示例1：大号粗体居中文字
```html
<p style="font-size: 20px; font-weight: bold; color: #0066cc; text-align: center;">
  重要提示文字
</p>
```

### 示例2：多行文字，不同样式
```html
<p style="font-size: 18px; font-weight: bold; color: #333; margin-bottom: 0.5rem;">
  标题文字
</p>
<p style="font-size: 14px; color: #666; line-height: 1.6;">
  描述文字，可以有多行内容。
</p>
```

### 示例3：带背景色的文字
```html
<p style="background-color: #f0f0f0; padding: 10px; border-radius: 5px; color: #333;">
  带背景的文字块
</p>
```

### 示例4：彩色高亮文字
```html
<p style="font-size: 16px; color: #ff6600; font-weight: bold;">
  橙色高亮文字
</p>
```

## 在 features 中的完整示例

```yaml
features:
  - title: 产品名称
    details: |
      <img src="/product.jpg" alt="产品" style="width: 100%; max-width: 300px; margin: 0 auto 1rem; display: block; border-radius: 8px;" />
      
      <p style="font-size: 18px; font-weight: bold; color: #0066cc; text-align: center; margin-top: 1rem;">
        产品标题
      </p>
      
      <p style="font-size: 14px; color: #666; line-height: 1.6; text-align: center;">
        这里是产品的详细描述文字，可以包含多行内容。
      </p>
```

## 常用颜色代码

- **黑色系列**：`#000000` (纯黑), `#333333` (深灰), `#666666` (中灰), `#999999` (浅灰)
- **蓝色系列**：`#0066cc` (标准蓝), `#0099ff` (亮蓝), `#003366` (深蓝)
- **红色系列**：`#ff0000` (纯红), `#cc0000` (深红), `#ff6600` (橙红)
- **绿色系列**：`#00aa00` (标准绿), `#009900` (深绿), `#66cc00` (亮绿)
- **紫色系列**：`#6633cc` (标准紫), `#9966ff` (亮紫)

## 提示

1. **使用内联样式**：在 Markdown 中使用 HTML 时，建议使用内联样式（`style=""`）
2. **单位**：字体大小可以使用 `px`、`em`、`rem` 等单位
3. **颜色**：可以使用颜色名称（如 `red`）、十六进制（如 `#ff0000`）或 RGB（如 `rgb(255,0,0)`）
4. **测试**：修改后刷新浏览器查看效果，根据需要调整数值
