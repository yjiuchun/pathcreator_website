# 示例页面 - 如何添加内容

这是一个示例页面，展示如何在 VitePress 中添加文字和图片。

## 📝 添加文字内容

在 Markdown 文件中，你可以直接使用 Markdown 语法编写文字：

### 标题
- `#` 一级标题
- `##` 二级标题  
- `###` 三级标题

### 文字样式
- **粗体文字**：使用 `**文字**`
- *斜体文字*：使用 `*文字*`
- ~~删除线~~：使用 `~~文字~~`

### 列表

无序列表：
- 项目1
- 项目2
- 项目3

有序列表：
1. 第一步
2. 第二步
3. 第三步

## 🖼️ 添加图片

### 方法1：使用 public 目录（推荐）

将图片放在 `docs/public/` 目录下，然后这样引用：

```markdown
![图片描述](/图片文件名.png)
```

**示例**：引用 public 目录下的现有图片
![搜索图标](/search.png)

### 方法2：使用相对路径

如果图片和 Markdown 文件在同一目录或子目录：

```markdown
![图片描述](./images/图片.png)
```

### 调整图片大小

使用 HTML 标签可以控制图片大小：

```html
<img src="/search.png" alt="搜索图标" width="200" />
```

实际效果：
<img src="/search.png" alt="搜索图标" width="200" />
